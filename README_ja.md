# 3D性格プロッター（3D Personality Plotter）

最終更新：2026年6月吉日　🍵

人間の行動傾向や性格を3次元座標空間にマッピングし、視覚的に分析するためのツールです。

本ツールは、**開発者自身の人付き合いへの苦手意識**という、きわめて個人的な動機から生まれました。自分の対人パターンをある程度客観的に観察し、他者との関係を少しでも改善するための自己デバッグ装置として設計されています。

---

## 🎯 主な用途

- 個人の内省・自己理解・自己デバッグ
- 歴史上の人物・文学作品・政治思想などの客観的な構造分析
- 心理学・行動生態学・組織論などの学術的な研究・教育目的での利用
